# Simple Web Server (SWS)

### About
Based on a UNIX domain server: http://www.linuxhowtos.org/C_C++/socket.htm

### Build and run
```
bash src/scripts/buildAndRun.sh
```

### Use with a navigator
```
http://{VM_ETH0_IP_ADDRESS}:8888/
```
